Expand description

Parse arguments into enums.

When deriving Clap, a field whose type implements ArgEnum can have the attribute #[clap(arg_enum)]. In addition to parsing, help and error messages may report possible variants.

Example

#[derive(clap::Clap)]
struct Args {
    #[clap(arg_enum)]
    level: Level,
}

#[derive(clap::ArgEnum)]
enum Level {
    Debug,
    Info,
    Warning,
    Error,
}
